# Semifields.jl

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://github.com/FJValverde/Semifields.jl/blob/main/LICENSE)

A m√©nagerie of semifields: semirings with a multiplicative group instead of a dioid. E.g. the tropical algebras, the algebra of probabilities, the entropic algebras, etc.
